# Tailwind CSS Alert - React
</DocsTitle>

Get started on your web projects with our Tailwind CSS <Code>Alert</Code> which provides contextual feedback messages for typical user actions.

An <Code>Alert</Code> displays a short and important message attracting the user's attention without interrupting the user's task. Use this element when you need to show highly-important messages to users or when you need a persistent static container that is closable by user actions.

See below our simple <Code>Alert</Code> example that you can use in your Tailwind CSS and React project. The example also comes in different styles and colors, so you can adapt it easily to your needs.

<br />

<CodePreview link="alert#alert" component={<AlertExamples.AlertDefault />}>
```jsx
import { Alert } from "@material-tailwind/react";

export function AlertDefault() {
  return <Alert>A simple alert for showing message.</Alert>;
}
```
</CodePreview>

<DocsTitle href="alert-variants">
## Alert Variants
</DocsTitle>

The <Code>Alert</Code> component comes with 3 different variants that you can change it using the <Code>variant</Code> prop.

<CodePreview link="alert#alert-variants" component={<AlertExamples.AlertVariants />}>
```jsx
import { Alert } from "@material-tailwind/react";

export function AlertVariants() {
  return (
    <div className="flex w-full flex-col gap-2">
      <Alert>A simple filled alert for showing message.</Alert>
      <Alert variant="gradient">
        <span>A simple gradient alert for showing message.</span>
      </Alert>
      <Alert variant="outlined">
        <span>A simple outlined alert for showing message.</span>
      </Alert>
      <Alert variant="ghost">
        <span>A simple ghost alert for showing message.</span>
      </Alert>
    </div>
  );
}
```
</CodePreview>
